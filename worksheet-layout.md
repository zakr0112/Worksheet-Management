?? Worksheet PDF Layout — Structural Breakdown
The worksheet PDF contains the following major sections (in order):

1. Header
Logo (left)

Header title (center)

Job number (right, large red font)

2. Job Details
A bordered table with:

Job Type

Job Date

Call Number

Contract Number

Engineers

3. Customer Details
Multiline block:

Customer name

Address

Postcode

4. Reason for Visit
Multiline block

5. Work Carried Out
Multiline block

6. Remedial Action / Variations
Multiline block

Referred To (single row)

7. Parts / Spares Used
Table:

Qty Used

Description & Size

8. Job Time / Travel Details
Table with:

Date

Travel Time

Arrived

Departed

Site Time

Job Time

Mileage

Totals row

9. Job Expenses
Table:

Date

Expense Details

Cost

Total Expenses

10. Customer Signature
Two modes:

No signature ? “NOT SIGNED”

Signature present ? Name + signature image

11. Photos
Up to 6 photos, placed:

Page 1: Photo 1 (top), Photo 2 (bottom)

Page 2: Photo 3 (top), Photo 4 (bottom)

Page 3: Photo 5 (top), Photo 6 (bottom)
