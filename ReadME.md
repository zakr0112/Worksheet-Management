# Worksheet Management Application

This repository contains a cross-platform worksheet management application developed using Delphi / RAD Studio.

## Project Overview

The application was created to replace a paper-based worksheet process with a digital alternative that supports:
- Customer and job record entry.
- Photo and signature capture.
- Local database storage.
- Worksheet output generation.
- Cross-platform use on Android and Windows.
